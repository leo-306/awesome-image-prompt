## 哪些词可以替换

**场景替换：**
- 露天茶饮店 → 咖啡馆门口 / 书店橱窗前 / 水果摊旁
- 草地仰拍 → 泳池边 / 沙滩 / 晒衣绳下
- 便利店 → 自动贩卖机前 / 夜晚路灯下 / 地铁出口

**胶片风格词替换（对应不同色调）：**
- `Fujifilm Pro 400H`（暖黄偏绿）→ `Fujifilm Superia 400`（冷调偏青）
- `Kodak Portra 400`（暖橘肤色友好）→ `Kodak Gold 200`（黄调复古）
- `CCD相机直拍感`（颗粒+过曝）→ `Lomo LC-A感`（暗角+漏光）

**道具替换：**
- 草莓奶茶 → 椰青 / 气泡水 / 柠檬茶
- 雪糕 → 棒冰 / 西瓜 / 便利店咖啡杯
