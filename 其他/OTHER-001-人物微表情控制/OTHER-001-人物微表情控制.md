# GPT Image 2 生图提示词｜其他 OTHER-001：人物微表情控制

图友们大家好，今天这一期是「人物微表情控制」。

很多人写人物情绪，只会直接写开心、伤心、震惊，结果出图很容易变成用力过猛的表演脸。这期把微表情拆成一套可以复制的公式：用眼部、嘴部、视线、肌肉松紧和光影氛围来控制情绪。

提示词主要按 GPT Image 2 的中文自然语言写法整理，也可以在豆包、千问及其他支持中文提示词的生图工具上尝试。不同工具对表情强度的理解不一样，可以微调镜头距离、光线和面部细节。

## 场景说明

这篇适合做人像表情图鉴、头像参考、情绪封面和 Prompt 教程配图。为了让模型更稳定，这一版统一成一个画面规则：一个场景、一个主体、上半身人物形象图。

统一基础结构是：

上半身人像 + 单人主体 + 浅色干净背景 + 眼部微动 + 嘴部微动 + 视线方向 + 柔和均匀光线

高级表情不是把情绪喊出来，而是让模型看到具体的生理反应。比如不要只写大哭，可以写「眼周微红、紧抿双唇、视线微垂」；不要只写震惊，可以写「眼睛微微睁大、嘴唇轻张、肩颈僵住」。

## 第一篇章：治愈系与心动感

高级的笑不是把嘴角拉到最大，而是让眼睛、嘴角和肌肉松弛度一起变化。

### 1. 窃喜 / 暗爽

适用场景：收到喜欢的人的消息、偷偷开心、少女心事但不过度外放。

提示词：

单人上半身人像，24 岁亚洲女生，浅米色纯色背景，黑色自然中长发，浅蓝色柔软针织衫，肩部到胸口构图，头部微微低下，嘴角想笑但忍住，轻轻咬住下唇，眼睛因为笑意微微弯起，视线略向下，柔和均匀棚拍光，真实皮肤质感，干净自然肤色，参考表情图鉴卡片风格，避免夸张大笑、多人同框、复杂背景、网红感和过度精修。

中文拆解：轻咬下唇 + 眼睛因笑意弯起 + 强忍笑意 + 柔和居家光线。

## 第二篇章：破碎感与故事感

这一类最容易出圈，但也最容易生成夸张哭戏。重点不要直接写「大哭」，而是写生理反应。

## 第三篇章：戏剧张力与高唤醒

突发情绪的关键不是把脸画得扭曲，而是让瞳孔、呼吸和肌肉僵硬感出现。

## 第四篇章：职场与松弛感

职场图不要只写精英感。更稳定的写法是用体态、眼神和浅表情，让人物看起来从容但不僵硬。

## 第五篇章：天真与生动

灵动感主要靠头部角度、身体前倾和眼神清澈度，不要把人物写成幼态。

## 第六篇章：极度反感与厌恶

厌恶感不是靠丑化人物，而是调动鼻翼、唇角和头部后撤这些真实反应。

单人上半身人像，24 岁亚洲女生，浅米色纯色背景，黑色自然中长发，浅蓝色柔软针织衫，肩部到胸口构图，头部轻轻向后缩，鼻翼微微皱起，眼睛眯起带着审视感，上唇一侧轻微卷起，像真实生理性排斥但不过度丑化，柔和均匀棚拍光，真实皮肤质感，参考表情图鉴卡片风格，避免夸张扭曲、多人同框、复杂背景、塑料皮肤和畸形五官。

![](image-11.png)

## 使用建议

1. 想更真实：优先写「眼周微红、嘴唇微张、下颚放松、眉毛微压」这类可见细节，不要只写抽象情绪。
2. 想做成图鉴感：统一使用浅色背景、上半身人像、单人主体和柔和均匀光线，只替换眼神、嘴角、眉毛和头部角度。
3. 想减少面部崩坏：在末尾加入中文负向约束，比如「避免夸张表情、卡通化、塑料皮肤、不自然肌肉紧绷、不对称眼睛、畸形瞳孔」。

建议收藏这篇。后续写人像 Prompt 时，只要替换人物设定、场景和光线，就能把同一套微表情公式复用到头像、封面、故事配图和职场人设图里。感兴趣的朋友们，欢迎收藏、关注，也可以在评论区留言你想看的表情或场景。
