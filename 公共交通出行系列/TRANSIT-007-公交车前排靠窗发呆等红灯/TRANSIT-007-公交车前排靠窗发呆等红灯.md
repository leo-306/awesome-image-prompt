图友们大家好，今天这一期是「公交车前排靠窗发呆等红灯」。

通勤路上有一种状态很难用语言描述——不是真的在看什么，就是对着窗外发呆，红灯、行人、灯箱招牌依次掠过，脑子是空的。这种瞬间拍起来特别有感染力，但很难靠摆拍复现，因为人一旦意识到在拍就会不自然。

用 AI 提示词可以绕过这个问题：把当时的光线、动作、情绪都描述清楚，让模型直接还原那一刻。

**这套写法的核心框架是：**「第一人称视角 + 公交车前排靠窗 + 发呆/低头/等绿灯 + 窗外路口环境虚化 + 光线类型 + iPhone 生活感抓拍」。

**可以替换的维度：**
- 光线：阴天冷光 → 晨光暖调 → 黄昏橙光 → 夜晚路灯透窗
- 动作：发呆望窗外 → 低头看手机 → 侧头靠窗枕着玻璃 → 转头看车厢内
- 焦段：35mm → 50mm → 24mm（广角更带空间感，50mm 更贴近人脸）
- 窗外：红灯路口 → 雨后湿路 → 夜晚灯箱招牌 → 清晨空旷街道

**适合哪些平台：** GPT Image 2 对中文自然语言理解最稳定，出图生活感强；千问也支持这类写法，整体偏暖调；豆包可以试试但建议加「iPhone 直出」来压住精修度。
