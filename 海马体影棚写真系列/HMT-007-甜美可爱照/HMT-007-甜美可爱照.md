## 这三条提示词里，有几个词特别值得拆出来说

**背景颜色要指定低饱和色**：浅粉、淡蓝、奶油黄，而不是直接写"粉色背景"——后者容易出现过饱和的糖果色，整体变廉价。

**姿态描述要具体到动作**：写"单手轻托腮"比写"可爱姿势"靠谱，动作越具体，AI 生成的稳定性越高。

**道具只选一个**：甜美风道具特别容易堆，小抱枕、发夹、帽子一起上，画面马上乱。保持单一道具，注意力集中在人物上。

**固定负向约束必须带上**：`避免AI美女脸、网红感、过度精修、塑料皮肤` 这一句是甜美风的核心控制，不写的话出图很容易漂移向商业美女写真方向。

---

## 这套写法可以替换哪些参数？

- **背景**：浅粉 → 奶油白 / 浅薄荷绿 / 浅米色，保持低饱和即可
- **服装**：泡泡袖 → 针织小开衫 / 碎花连衣裙 / 白色衬衫
- **道具**：小抱枕 → 花束 / 发夹 / 毛绒玩偶（一次只放一个）
- **发型**：半扎发 → 双马尾 / 自然卷发 / 清爽丸子头
