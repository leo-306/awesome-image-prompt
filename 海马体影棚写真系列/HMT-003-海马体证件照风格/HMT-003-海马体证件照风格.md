很多图友反馈：用 AI 生成的证件照，总觉得差点意思——背景是白的，人也是正面的，但就是没有那种「花了三百块去海马体拍出来」的感觉。

问题出在哪？今天这一期，我用三个版本的 Prompt，把这个问题从头到尾演示一遍。

---

**Q：第一版，最常见的写法，出来什么效果？**

大多数人第一次尝试证件照提示词，写的都是这样的：

<div style="background:#f0faf0; border-left: 3px solid #a8d5a2; padding: 12px 16px; border-radius: 4px; margin: 8px 0; font-size: 15px; line-height: 1.8; color: #333;">
一张证件照风格的人物照片，24岁亚洲女生，穿深色西装，白色背景，正面朝镜头，微笑，五官自然清秀，面部干净，健康自然肤色，避免 AI 美女脸、网红感、过度精修、塑料皮肤、暗沉肤色、明显痘印、明显皱纹、斑点、面部变形
</div>

![](image-01.png)

> **结果：** 出来了，但很普通。面部光线非常平，没有立体感；背景是硬白，没有质感；整体更像随便对着手机拍了一张。跟海马体差距明显。
>
> **缺了什么：** 布光词、背景质感词、妆容细节、发型描述、镜头焦段——这些全没有。

> **结果：** 好多了。光线有了层次感，背景也柔和了。但妆容描述不够精细——「韩式精英通勤妆」这个词太模糊，AI 有时候会往「网红精致感」走偏，出来的五官还是有点过于完美。
>
> **还差什么：** 妆容细节（眉形、腮红颜色、唇色）、背景渐变质感、镜头焦段、肤质控制——精确到位才能复刻海马体那种「精修但自然」的度。

## 三个版本改动了什么

| 维度 | Version 1 | Version 2 | Version 3（最终版） |
|---|---|---|---|
| 布光 | 无 | 蝴蝶光 | 蝴蝶光 + 45度方向 + 鼻梁/颧骨描述 |
| 背景 | 白色 | 浅灰低饱和 | 淡米白 + 轻微柔焦渐变 |
| 妆容 | 无 | 韩式精英通勤妆（模糊） | 一字眉 + 肤感底妆 + 珊瑚腮红 + 裸色唇 |
| 发型 | 无 | 黑长直高颅顶 | 盘发 + 高颅顶 + 碎发细节 |
| 镜头 | 无 | 无 | 50mm 标准镜头 + 轻微景深 |
| 肤质控制 | 基本负向约束 | 通透自然 | 通透细腻有光泽 + 眼神光高光 |

---

最终版可以直接复制，也可以把「西装+白衬衫」换成「浅色针织」走学生风，或者换「印花衬衫」走文艺生活照风格——背景和布光词保持不变，妆容可以微调腮红颜色。

海马体的核心不是哪一个词，是所有维度同时对准：布光 × 背景质感 × 妆容细节 × 肤质控制，缺一个都会掉档。
