# 自然奇观环游系列选题大纲

## 系列定位

**系列名称：**《她逃向世界尽头》——24 岁亚洲女生自然奇观环游记  
**系列编号：**`WILD`  
**核心主题：**自由、自然、远离尘嚣。固定主角暂时离开喧嚣生活，只身走向海岛、草原、群山、荒漠和极地，在辽阔天地中重新找回自由。  
**内容边界：**只呈现自然奇观与旷野旅行，不混入城市街道、商业空间、拥挤游客和明显现代建筑。

## 固定主角设定

- 24 岁漂亮亚洲女生，清秀自然的东亚面孔，黑色长发，身形纤细健康。
- 淡妆或近乎素颜，保留皮肤纹理，不过度磨皮，不做影楼式摆拍。
- 长途旅行中保持同一张脸、相近发型和自然气质；服装随气候变化，但以白、米色、浅蓝、卡其和大地色为主。
- 常用动作：迎风站立、赤脚涉水、沿小径行走、奔跑、回头、远眺、张开双臂、坐在旷野发呆。
- 常用镜头：环境人像、人物占画面较小、广角远景、背影与侧脸、自然抓拍、电影感构图。
- 固定情绪：松弛、自由、安静、敬畏天地，避免夸张笑容和商业模特感。

## 微信文章标题策略

标题不使用固定模板。生成每篇文章时，先读取同系列最近 3 篇 `run-state.json` 的 `title_style`，选择未重复的句式；标题控制在 24—38 个中文字符，最长不超过 45 个字符，不出现具体 AI 工具品牌名。

每个标题至少突出一个读者能直接感知的核心：**景点奇观、女友感、出图效果或生成方法**。可以提“提示词”，但不要每篇都写；也可以改用“这套写法”“生成方法”“一行字”等普通读者更容易理解的表达。

可轮换的标题方向：

| 句式类型 | 示例 |
| --- | --- |
| 景点直给型 | 马尔代夫玻璃海太出片了，随手生成就是旅行大片 |
| 女友感结果型 | 把女友放进塞舌尔海滩，这组自然旅行照太松弛了 |
| 提示词分享型 | 撒哈拉日落提示词，这套金色沙丘写法可以直接复制 |
| 生成方法型 | 不用真的去冰岛，这套写法也能生成蓝冰洞旅行照 |
| 疑问引导型 | 为什么雪山女友照总像游客照？关键要改这几个词 |
| 效果对比型 | 同一个峡湾换 3 种构图，哪张更有世界尽头感？ |
| 故事悬念型 | 她离开城市后的第一站，是一片没有尽头的玻璃海 |
| 社交传播型 | 朋友圈都以为她去了南极，其实只写了这一段话 |

## 目录状态规则

- `是否完成` 为空：该选题尚未完成文章、封面和图片生产。
- `提示词` 为空：尚未创建文章 Markdown，工作流优先处理第一条空记录。
- 创建文章后，`提示词` 更新为文章的相对链接。
- 文章、封面和图片全部落盘后，`是否完成` 更新为 `是`。
- 更新状态时只修改当前选题行，不重排其他行。

## 低成本连载策略

1. 每周更新 3 期，按编号顺序推进，让读者明确感受到旅程进度。
2. 每期使用“1 张封面主图 + 3—5 张同场景变化图 + 完整 Prompt”的固定结构。
3. 同一章节共用一套基础 Prompt，只替换地点、地貌、天气、动作和服装。
4. 每 10 期发布一次章节合集；WILD-030 和 WILD-060 分别制作半程、全程总目录。
5. 封面固定显示“她逃向世界尽头 + WILD 编号 + 地点”，强化系列识别。
6. 每篇结尾预告下一站，以连续故事提高关注和复访。

## 转化文案模板

**关注引导：**她正在用 60 站逃向世界尽头。关注我，陪她继续走过雪山、荒漠与极地。  
**收藏引导：**收藏这期，后续可以直接替换地点和服装，复用同一套自然旅行 Prompt。  
**点赞引导：**如果你也向往这种远离尘嚣的自由感，点个赞，我会继续更新下一站。  
**下期预告：**下一站：`WILD-XXX｜具体场景`。  
**合集引导：**每 10 期整理一次章节合集，回复“世界尽头”查看完整目录。

## 后续生产规则

- 每期必须保持固定主角身份和面部特征，优先使用角色参考图保证一致性。
- 人物、动作、空间、时间、光线和情绪六个要素必须完整。
- 地标只作为自然环境背景，人物与环境都要有足够画面占比。
- 避免城市天际线、道路车辆、酒店、商店、密集游客和商业摄影棚质感。
- 不表现无防护攀爬、靠近熔岩、悬崖探身等危险行为；极端环境明确使用安全距离与合适服装。
- 单期正式 Prompt 在进入生产时再展开，本目录只维护选题、故事和提示词要点。
