# 山海神铸：6位东方英雄，被做成古籍水墨电影海报

<div style="text-align:center; margin: 0 0 24px 0;">

![](image-08.png)

</div>

这组作品被命名为「山海神铸」。六位英雄来自神话、传奇与历史，却被放进同一套视觉系统：旧宣纸是时间，巨大人物轮廓是纪念碑，轮廓内部的山河、城池与人群则是人物一生承担的世界。画面不靠金甲、爆炸和满屏特效制造“强”，而是用<span style="color:#d97757; font-weight:600;">巨大侧影与微小人物的尺度反差</span>，让英雄的选择显得更重。

这套海报最重要的设计原则，是<span style="background:#fff3cd; padding: 1px 4px; border-radius: 2px;">先让每个人物拥有独立叙事，再用统一材质把六张图收成一套</span>。焦墨、深靛、暗朱红与旧金负责系列识别；海浪、云气、飞沙、星屑和纸张纤维，则根据人物命运发生变化。

第一张从哪吒开始。这里不是可爱的儿童形象，也不是复制某个熟悉的银幕造型，而是把他理解成“站在海与人间之间的少年守护者”。巨像负责精神压力，礁石上的小人物负责行动，火与水构成整张海报最直接的视觉冲突。

**哪吒·踏海焚天｜原版提示词**

<div style="background:#f0faf0; border-left: 3px solid #a8d5a2; padding: 12px 16px; border-radius: 4px; margin: 8px 0; font-size: 15px; line-height: 1.8; color: #333;">竖版3:4，中国神话英雄主题艺术海报，原创东方水墨双重曝光构图，哪吒人物意象。一位少年神将的巨大半身侧影位于画面右侧，身体微微向左转，头部昂起看向远方，面容清秀坚定，眉眼带有少年锐气，黑色长发束成高位双髻，系暗红色长绫，额前与鬓角保留被海风吹动的碎发。人物造型融合古代神话少年将军、红绫、金色护腕、轻型东方战衣和火焰纹样，服饰原创设计，不复刻任何具体动画电影角色造型。人物头发、面部暗部、肩膀和披风内部采用水墨双重曝光，嵌入一整片翻涌东海：黑色巨浪、海上风暴、断裂礁石、云层、雷电、海底龙宫、青铜宫门、盘旋龙影、被火光照亮的海面。人物额头内部出现一轮暗红色太阳，胸口内部是一座被海水包围的龙宫大殿，肩部内部出现极小的百姓、渔船与海岸城镇。画面下方，一名体量极小的少年英雄赤足站在黑色礁石顶端，手持火尖枪，混天绫在风中向后飞舞，脚下隐约浮现燃烧的风火轮，正面对抗远处掀起巨浪的龙形云雾。人物与巨大侧影形成强烈尺度反差，体现少年反抗、命运选择与守护苍生的史诗感。背景为温暖米白色旧宣纸，大面积留白，纸面带纤维、轻微泛黄、水渍、墨迹和自然斑驳。主体颜色以焦墨黑、海水青灰、深靛蓝、暗朱红、赭石金和少量火焰橙为主，红色只集中在混天绫、火焰与印章区域。海浪、发丝和披风边缘逐渐溶解成泼墨、飞白、烟雾与火星颗粒。画面左侧预留竖排毛笔书法标题区域，书法笔势凌厉有力，搭配纤细小楷题跋、暗红色朱砂篆刻印章和底部小号古典衬线英文标题，不生成乱码文字。东方神话、古籍装帧、电影级水墨概念艺术、巨大尺度、浪涛纵深、火与水对撞、少年英雄、庄严热血、原创视觉设计。负面提示词：复刻具体动画电影哪吒造型，丸子脸婴儿形象，现代运动鞋，现代服装，欧美超级英雄，赛博朋克，科幻机械，卡通Q版，低龄儿童画，夸张邪笑，网红脸，过度磨皮，塑料皮肤，不合时代的轻浮服饰，火焰过度饱和，龙体结构错误，多余手臂，长枪弯曲，风火轮粘连，硬边拼贴，纯黑背景，文字乱码，logo，二维码，平台水印。</div>

![](image-01.png)

这份原版写法看起来很长，但结构其实清楚：先定人物姿态，再决定双重曝光里装什么，接着安排画面下方的“行动者”，最后统一纸张、颜色、留白和书法区域。<span style="color:#d97757; font-weight:600;">人物—内景—小人物—材质</span>，是整套海报可以反复复用的四层骨架。

孙悟空这一张把“火水对撞”换成了“云宫动荡”。人物轮廓从右侧换到左侧，金箍棒形成左下至右上的斜线，打破哪吒海报的纵向稳定。真正的核心并非猴王的武力，而是<span style="background:#fff3cd; padding: 1px 4px; border-radius: 2px;">一个人面对庞大秩序时仍然向前</span>：南天门越高、断阶上的行者越小，自由意志就越清晰。

![](image-02.png)

花木兰的处理刻意收住战斗场面。她的内景不是一场胜利，而是一条从纺织机、旧盔甲、军营走向雪山关隘的长卷。心口那间亮着灯的小屋，是整张图的情感锚点。冷蓝风雪与室内暖光被分到人物两侧，让“离家”和“守家”同时存在。<span style="color:#d97757; font-weight:600;">英雄感不等于愤怒表情</span>，克制的回望往往更有力量。

![](image-03.png)

关羽的视觉主题不是挥刀，而是远行。长髯、偃月刀弧线和山间古道彼此呼应，三种曲线把人物、兵器和旅程连在一起。远处开启的城门没有交代结果，却制造了“下一关即将到来”的悬念。这里用深青绿替代常见的高饱和绿色，也避开夸张脸谱，忠义因此更像一种沉默的承诺。

![](image-04.png)

到了诸葛亮，画面的速度再次降低。风本来不可见，所以让它通过转向的船帆、扬起的宽袖、飞散纸页和长墨线被“看见”。星象图放在额头，江岸祭坛落在胸口，理性推演与现实局势形成两层空间。<span style="background:#fff3cd; padding: 1px 4px; border-radius: 2px;">不要给“智慧”加魔法光效，让环境对人物的判断作出反应</span>，画面会更高级。

![](image-05.png)

岳飞作为收束，用山河代替奇观。披风像山脉，甲胄内部是城池、军阵和归家的百姓，右上方晨光只照亮眼睛、盔甲边缘与远处河流。它没有把遗憾写成阴郁，而是留下一束暴雨后的光。人物凝视的是画外远方，观者会自然追随视线，把未完成的故事补下去。

![](image-06.png)

六张图虽然共享水墨双重曝光，但每张的“运动方式”不同：

| 人物 | 内部世界 | 主导动势 | 情绪关键词 |
| --- | --- | --- | --- |
| 哪吒 | 东海、龙宫、海岸百姓 | 火与浪对撞 | 选择、守护 |
| 孙悟空 | 天宫、雷云、花果山 | 斜向冲破 | 自由、反抗 |
| 花木兰 | 家园、军营、雪关 | 冷暖迁移 | 责任、归乡 |
| 关羽 | 驿道、渡口、城门 | 曲线远行 | 守信、忠义 |
| 诸葛亮 | 江面、战船、星象 | 风线流动 | 判断、克制 |
| 岳飞 | 城池、军阵、山河 | 纪念碑式静止 | 故土、未竟 |

如果要和 AI 继续打磨这类作品，最有效的交互不是一句“再高级一点”，而是分三轮下指令。第一轮只看<span style="color:#d97757; font-weight:600;">人物身份和大构图</span>，确认巨像位置、视线方向与小人物是否形成尺度差；第二轮检查双重曝光的内景有没有叙事顺序，删掉抢戏或重复的建筑；第三轮再处理纸张纤维、墨迹溶解、局部朱砂红和留白。

可以直接这样反馈：“保留人物姿态和宣纸底不变，把胸口内景从杂乱建筑改成一条由近及远的叙事路径；红色只保留在印章与关键道具；增强内景与人物轮廓的自然溶解，避免硬边蒙版。”<span style="background:#fff3cd; padding: 1px 4px; border-radius: 2px;">一次只改一个层级，模型更容易保住已经正确的部分。</span>

这套方法也能迁移到其他人物：先问“他真正守护什么”，再问“什么场景能被装进他的轮廓”，最后才决定兵器、服装与特效。这样得到的不是一张角色立绘，而是一幅能被读出前因后果的视觉传记。
